تقویم شمسی حرفه‌ای با شورتکد، پنل مدیریت پیشرفته، ابزارک، بلوک گوتنبرگ و ویجت المنتور.

== Installation ==
1. فایل zip افزونه را در وردپرس آپلود کنید.
2. افزونه را فعال کنید.
3. از منوی Persian Calendar Pro تنظیمات را انجام دهید.
4. شورتکد [persian_calendar_pro] را در صفحه قرار دهید.

== Features ==
- مناسبت‌های ایران + جهانی + قمری (پیش‌فرض/سفارشی)
- اوقات شرعی خودکار بر اساس شهر
- تم‌های Material / Minimal / Dark
- سفارشی‌سازی کامل رنگ‌ها، فونت، CSS، ریسپانسیو
- مدیریت رویدادها با Drag & Drop
- Gutenberg block + Elementor widget + WordPress widget
- JSON import/export
- Cache برای سرعت
